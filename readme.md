# The Family Tree App

A cross-platform family tree application built with Tauri, React, and React Flow.

Work in progress.


![Web](screenshot.png)

![Tauri](screenshot2.png)

## Tech Stack

- **Frontend**: React + React Flow
- **Desktop**: Tauri (Rust)
- **Web Database**: IndexedDB
- **Desktop Database**: SQLite
- **Build Tool**: Vite
